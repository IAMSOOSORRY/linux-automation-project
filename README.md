# Linux Automation Project

## Objective
Automate Linux administration tasks using Bash scripts.

## Features
- Daily log backup
- CPU monitoring
- Memory monitoring
- Disk monitoring
- Automated cleanup of old backups
- Cron job scheduling

## Scripts
- backup.sh
- monitor.sh
- cleanup.sh

## Technologies Used
- Linux (Ubuntu)
- Bash
- Cron
- Git & GitHub
